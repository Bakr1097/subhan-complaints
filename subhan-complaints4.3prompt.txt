BACKGROUND ON CALLMEBOT
Callmebot is a free service that sends WhatsApp messages via a simple GET 
request. Each recipient must register once by saving the Callmebot number 
to contacts and sending it an activation message — this returns a unique 
API key per number. I will do this registration manually and provide you 
the two phone numbers and their two API keys to put in environment 
variables. Do NOT hardcode them.

The Callmebot send URL format is:
https://api.callmebot.com/whatsapp.php?phone=[PHONE]&text=[URL_ENCODED_TEXT]&apikey=[APIKEY]
